TFT-LCD enabled energy mode demo.

This example project uses the EFM32 CMSIS libraries and demonstrates  the 
use of the EBI controller, RTC (real time counter), GPIO and various Energy 
Modes (EM).

First press the kits AEM button to give display control to the EFM32. Then
use joystick up and down to select the wanted energy mode demo. Press PB1
to activate the demo. Press the AEM-button to go back to the board control
screen to view the AEM current display.

Board:  Silicon Labs EFM32-G2xx-DK Development Kit
Device: EFM32G290F128 without LCD (with EBI)  _ONLY_
